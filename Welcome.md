# Welcome to the Shared Vault

This vault comes pre-configured with:

## Plugins
- **Claude Threads** — AI conversation threads inside Obsidian
- **Google Docs Sync** — sync notes to/from Google Docs
- **Linear Integration** — sync Linear issues into your vault
- **Kanban Bases View** — kanban boards for your notes

## Getting Started
1. Open this vault in Obsidian (File → Open Vault)
2. Obsidian will prompt you to install community plugins — click **Install**
3. Trust the plugins when prompted
4. Configure each plugin with your credentials (see [[Plugin Setup]])

## Daily Notes
Daily notes live in `Daily/` — use the calendar plugin or Cmd+P → "Open today's daily note".
